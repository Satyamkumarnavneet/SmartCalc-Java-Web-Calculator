# SmartCalc - Java Web Calculator

SmartCalc is a versatile and user-friendly web calculator developed in Java. This application provides a dynamic platform for performing basic arithmetic operations, including addition, subtraction, multiplication, and division. With an intuitive user interface and responsive design, SmartCalc allows users to input numerical values and select the desired operation effortlessly.

## Key Features:

- **Arithmetic Operations:** Perform addition, subtraction, multiplication, and division with ease.
- **Error Handling:** Robust error handling for invalid inputs, preventing unexpected issues during calculations.
- **Responsive Design:** User-friendly interface that adapts to various screen sizes, providing a seamless experience on both desktop and mobile devices.
- **Real-time Results:** Instantly view the calculated result without the need for page refresh, enhancing user interaction.
- **Visual Appeal:** Stylish and modern design with a pleasant color scheme for an aesthetically pleasing user experience.

## Tech Stack

**Backend:**
- Java
- Servlets
- JSP (JavaServer Pages)

**Frontend:**
- HTML, CSS
- JavaScript
- Bootstrap (for styling)

**Deployment and Hosting:**
- Apache Tomcat (or any other Java servlet container)
